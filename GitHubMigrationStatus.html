GitHub Migration Status

The LLVM project is migrating its source code from SVN to git. This page tracks the current status of the migration.

We plan to complete the migration by the 2019 US Developers Meeting (Oct 21, 2019).
You should begin migrating your workflows from SVN to GitHub today.

Migration Instructions For:
  • CI or other automated workflows: Start pulling code from https://github.com/llvm/llvm-project instead of SVN.
  • Developers: Use the git-llvm script for committing changes.
