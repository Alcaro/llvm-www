Windows snapshot builds

We provide a periodically updated installer for Windows. The current installer is built with MSVC 2012 and targets the MSVC 2010 C/C++ runtimes.

Windows installer (mirror), based on SVN r190067.

Currently, the package only includes clang and the Address Sanitizer runtime from compiler-rt, but eventually it should grow to encompass lld, lldb, and possibly other LLVM projects such as libc++.
